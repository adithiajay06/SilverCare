# SilverCare
“AI-powered elderly care system with voice assistance, fall detection, and caregiver monitoring”

## Features
- Voice-based medicine reminders
- AI-powered fall detection
- Real-time location tracking (for dementia patients)
- Instant alerts to caregivers
- Voice-based interaction system
- Smartwatch integration

## Technologies Used
- Flutter / Android
- Firebase (Backend & Database)
- AI/ML (Fall detection, Voice recognition)
- Google Maps API
- Wearable APIs (Smartwatch integration)

## Problem
Elderly individuals often face risks like missed medications, falls, and lack of immediate assistance.

## Solution
SilverCare provides a voice-first, AI-driven system that ensures real-time monitoring and caregiver connectivity.

## Future Scope
- Predictive health analytics
- Smart home integration
- Doctor connectivity
